Getting the sample dashboards

You can download sample Splunk and Elastic dashboards that present data forwarded by Workbench.

Splunk dashboards

The sample Splunk dashboards are available in a Splunk app. You can get the app in two ways:

  • From Docker Hub, in a Docker image that contains an installation of Splunk configured with the app and sample data.

    This option is useful for demonstrating the dashboards. You can start a Docker container, and then immediately use the dashboards with the supplied sample data, with no configuration required.

  • From GitHub, as an app directory that you can add to your own Splunk installation.

Elastic dashboards

The sample Elastic (Kibana) dashboards are available from Docker Hub in a Docker image that contains an installation of the Elastic Stack configured with the dashboards and sample data.
