atf-aes: OMEGAMON for IMS ATF

This JCL forwards IBM OMEGAMON for IMS on z/OS Application Trace Facility (ATF) summary log records that have been written to the IMS log.

You need to customize this JCL to specify the IMS log code that ATF used to write those records to the log. You need to specify that log code in two places in the SYSIN data set:

  • The ATF command
  • The CODE parameter of the JSON command
